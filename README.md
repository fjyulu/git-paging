## Git Paging

Treat git log as a book, exec `git next` or `git prev` to checkout the next or the previous commit.

### 像翻页一样跳转到上一(n)条或下一(n)条 Git 历史节点

请查看博客文章 [阅读开源代码小技巧](https://hutusi.com/git-paging) 获得更多使用帮助。
